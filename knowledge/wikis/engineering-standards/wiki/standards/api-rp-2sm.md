# API RP 2SM Synthetic Fiber Rope Mooring

Metadata-only citation resolver for `digitalmodel` synthetic fiber rope
mooring helpers. The implemented subset covers material selection, axial
stiffness screening, creep screening, axial-compression fatigue screening, and
manufacturing / installation QA checklist items for the 2001 recommended
practice with the 2007 addendum.

This page intentionally does not reproduce standard text or tables. The source
standard remains the controlling reference for project rope qualification,
manufacturer data, installation procedures, and acceptance criteria.
