## 1. GT1R R35 Bolt-On Parachute Kit

**Manufacturer**: T1 Race Development
**Product URL**: https://www.t1racedevelopment.com/product/gt1r-r35-bolt-on-parachute-kit/

- **Design**: Single parachute only

### Mounting
- Bolts to frame rails at original aluminum bumper beam location
- Tag line and chute support extends behind license plate

- Kit includes: large pilot chute, 430 canopy, bridle cord, deployment bag, 8.75" aluminum plate, pack

| Configuration | Speed Range | Recommended Model |
|---|---|---|
| Single chute | Up to ~200 MPH | 430 Std. 32 |
| Dual chute | 200–260 MPH | 430-24 Pro Stock |
| Dual chute | 240–320 MPH | 430-26 / 450 |
| Dual chute | 280–320 MPH | 450 / 470 |

is designed for single chute only, the 250 MPH load case represents an
extreme/conservative scenario.
