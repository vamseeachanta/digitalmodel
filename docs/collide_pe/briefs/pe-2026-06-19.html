PE Problem of the Day 6/19/2026 — Productivity Index

productionupstreamreservoir-engineeringproductivity-index

Problem

A vertical oil well is producing from a sandstone reservoir. The well has a productivity index (PI) of 1.5 BBL/PSI-D. The reservoir pressure is 2,800 psi and the current flowing bottomhole pressure (FBHP) is 1,600 psi. Q1: What is the current oil production rate? Later, a small tubing modification reduces FBHP to 1,400 psi. Q2: What is the new oil rate after the modification?

Given

productivity index bbl per psi d1.5
reservoir pressure psi2800
fbhp initial psi1600
fbhp after mod psi1400

Find

  • Q1: current oil production rate (STB/d)
  • Q2: oil rate after FBHP drops to 1,400 psi (STB/d)

Method

Straight-line IPR (Productivity Index): q = PI x (Pr - Pwf), valid above the bubble point.

Worked Solution

  1. Drawdown_1 = Pr - Pwf_1 = 2800 - 1600 = 1200 psi
  2. Q1 = PI x drawdown_1 = 1.5 x 1200 = 1800 BBL/d
  3. Drawdown_2 = Pr - Pwf_2 = 2800 - 1400 = 1400 psi
  4. Q2 = PI x drawdown_2 = 1.5 x 1400 = 2100 BBL/d

Answer

q1 bbl per day: 1800
q2 bbl per day: 2100
delta bbl per day: 300

Engineer's Note

The linear PI model assumes single-phase flow above the bubble point and a constant PI. If FBHP falls below the bubble point, use Vogel's IPR instead — rate gains taper off rather than staying linear.
