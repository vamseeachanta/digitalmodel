Known Formulation Defects

Defect 1: Sway magnitude ~98x too high

Root cause: _calculate_infinite_depth() uses max(L1,L2) as single L; A_midship = B·T·Cb (single area) instead of A&sub1;×A&sub2;; missing 1/(2π) factor.

Impact: Absolute force values unreliable; relative comparisons valid.

Defect 2: Surge non-zero at abeam

Root cause: g_kernel uses s1(ξ)·ds1(η) — passing vessel uses undifferentiated S&sub1; instead of dS&sub1;/dx; integral not antisymmetric in stagger.

Impact: Surge values at all stagger positions are suspect.

Defect 3: Surge not antisymmetric

Root cause: Same kernel issue; Wang requires dS/dx for both vessels and 1/r² (not 1/r³).

Impact: Force symmetry properties incorrect for surge.
