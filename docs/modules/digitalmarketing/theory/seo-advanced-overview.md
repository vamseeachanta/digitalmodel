    - IAAT (In Anchor and Title) Numbers. Competition is using the keyword in page title and text link i.e. competitive keyword
    - Competition Index 
    - Keyword Effectiveness Index (KEI). Number of monthly searches to number of competitive websites. 
    - Keyword Trends i.e. monthly trends
    - SERP preview: Top 10 sites for a keyword

    - most popular keyword is not always best. Less popular keyword can help raise website visibility faster
    - Pareto principle helps in choosing less popular keywords
    - short tail (1 to 2 words). 
        - Typically the top 3 or 5 most popular ones
        - Early stage of buying cycle. Long established companies or bigger budget companies can utilize these.
    - long tail (3 or greater). 
    - In conclusion
        - Understand the search pathway

            - study keyword research

## On-page Optimization Techniques

- How to implement keywords into the content
- What are the key page elements:
    - Page title
        - make it clear and simple
        - use keywords or keyword phrases relevant to the content
        - do not add additional words than necessary
        - shorter titles for better readability
    - H1 and H2 important for both search engines and humans
        - H1 is the headline. Short, concise and clear
        - H1 is written in active voice
        - Avoid nuance or joke as it is confusing
        - Goal is to describe the page content
- How to structure the content
    - Breakdown into hierarchy of information
    - Useful for users (and search engines to determine relevancy)
    - Visual appeal is important. Properties include
        - typography: hierarchy, font type, weight, size
        - layout: placement of images and text
        - fontsize: style
        - color scheme: good combination
    - Not necessarily lot of artistic elements.
    - Any page should have unique url, page title, description and H1 headline
    - Bad designs:
        - Having too many calls-to-action.
        - Too much clutter
        - redundant information
        - competing or canibalizing each other
        - content is odd and unnatural
    - optimization should be natural practice of developing content in a way to answer questions
    - communicate content in your page in less than 8 seconds. Utilize HTML and seo techniques to make content more scannable
        - H1
        - H2
        - bullet points (20% more read them)
        - hyperlinked text
        - contrasting colors : eg. bold black with white space
        - docs\ref\on_page_seo_html_layout.PNG
    - Designing website
        - docs\ref\on_page_seo_url_website_structure.PNG
        - keywords for unique page titles
        - divide by dashses
    - HTML5
        - Provides multiformat (i.e. microformats)
        - microformats help develop content. Tags in microformat inform search engines and machine learning applications
        - microformats: contact card, calendar events, review, article
- Optimize for mobile
    - Google maintains seperate index for Mobile as mobile is a primary way i.e. mobile first index
    - desktop is still signficant and more conversion (complete transactions)
    - Conduct a mobile friendliness test
        - https://search.google.com/test/mobile-friendly
        - best way is to access url and do render tests
    - utilize responsive design i.e. using view port
    - Use the Content management system (CMS) to test for all devices
    - dynamic page serving: served based on device (Not recommended)
        - not recommended. Takes immense programming.
        - playing catch up
        - search engines may get confused
    - Create a mobile website
        - More work to maintain and optimize

    - 2 urls to the same url (good)

        - makes difficult for the search engines to figure out the main version
        - utilize 301 redirect (helps it)

    - caused by some CMS systems

            - Alternatively utilize Robots.txt (more work)
