Production engineering — nodal analysis & well deliverability

A node-based well-performance engine: intersect an inflow (IPR) curve with a vertical-lift (VLP) pressure traverse to find the operating point, then bound it with a confidence band derived from the underlying test-quality score. Distinct from the rod-pump dynacard diagnostics under Artificial lift.
