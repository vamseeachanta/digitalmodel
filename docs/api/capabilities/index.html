Tubular products

Material grade matrix

API 5L / ISO 3183 · IACS UR W11 · EN 10025-2
  • 33 reconciled grades, single source of truth
  • PSL1 vs PSL2, US-customary vs ISO resolved
  • back-compat adapters for legacy dicts

Line pipe catalog

API 5L · ASME B36.10M
  • NPS 1/8–80 × schedule × grade → LinePipe
  • ID, D/t, metal area, plain-end weight, bore volume

Casing & tubing

API 5CT · API 5C3 / TR 5C3
  • 4-regime collapse (yield/plastic/transition/elastic)
  • internal-yield burst, pipe-body yield
  • feeds the VME / API-ellipse design envelope

Sucker rod

API 11B · modified Goodman
  • grades C/K/D, sizes ½–1¼ in
  • service-factor (sour) derated allowable stress
  • ties into the dynacard rod-buckling workflow

Ship structural strength

Hull-girder longitudinal strength

IACS UR S11 / S11A
  • wave bending moment + still-water envelope
  • section-modulus yield check (175/k)
  • simplified + full Smith incremental-iterative ultimate

Plate & stiffened-panel buckling

DNV-RP-C201
  • plate-induced, column, torsional (tripping)
  • Johnson-Ostenfeld inelastic correction

Grillage & girder / web-frame

DNV-RP-C201 · DNV CN 30.1
  • orthogonally-stiffened grillage (overall mode)
  • web-panel shear buckling; tripping-bracket spacing

Corrugated bulkhead

IACS CSR · DNV-RP-C201
  • trapezoidal-corrugation section properties
  • flange/web plate & overall buckling screens

Class-rule scantlings

DNV-RU-SHIP · IACS CSR
  • minimum plate thickness by location
  • plate-for-pressure & stiffener section modulus

Fitness-for-service & fatigue

Corroded-pipe remaining strength

ASME B31G · DNV-RP-F101
  • B31G / Modified / RSTRENG effective-area
  • DNV-RP-F101 single + interacting (length-weighted)
  • full Part-A PSF tables + validity layer

Metal-loss / level FFS

API 579-1 / ASME FFS-1
  • GML/LML Level-2 RSF, Folias factor
  • measurement-sufficiency & decision layer

Circumferential & 2D defects

API 579 Part 5 · RSTRENG
  • circumferential net-section RSF
  • 2D river-bottom (axial×circumferential grid)

Connection & tubular-joint SCF

DNV-RP-C203 (Efthymiou)
  • T/Y SCFs (full-fidelity + short-chord F1)
  • weld-detail hot-spot stress + S-N class

Provenance

Every strength / FFS result carries a code_reference naming its governing code and edition, sourced from one codes.py register that the codes register documents — a CI test keeps the two in sync, so a result can never cite a code the register doesn't list.
