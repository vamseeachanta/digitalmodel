1  Objective

Verify the digitalmodel CFD pipeline for a lift-generating body — a new physics regime beyond the drag/friction and vortex-shedding cases in the suite. The benchmark is the classic NACA0012 airfoil, whose lift-curve slope in the linear (pre-stall) regime is set by thin-airfoil theory and well documented experimentally.

thin-airfoil theory:  dCl/dα = 2π per radian = 0.1097 per degree

Real airfoils fall slightly below 2π because of finite thickness and the viscous displacement effect; the measured NACA0012 slope is ≈0.106/deg (Abbott & von Doenhoff).

5  A note on the reference frame

Zero-lift angle recovered as a check. The mesh (the OpenFOAM airFoil2D NACA0012 C-grid) carries a small built-in geometric incidence; the fit recovers it as a zero-lift angle of -1.4°. Because a rigid incidence offset shifts the curve's intercept but not its slope, the lift-curve slope — the verification target — is unaffected. Forces are resolved in wind axes (drag along the free stream, lift normal to it), and the reference area uses the measured chord (≈35, from the walls patch) times the span, so the coefficients are correctly normalised.
