Normalized master curve

Dimensionless sloshing frequency Ω₁ = ω₁√(Lc/g) vs fill / slenderness ratio h/Lc

Any tank of a given shape — any size — lies on its curve. Read Ω₁ off the curve at your fill ratio, then the real period is T = 2π√(Lc/g) / Ω₁. Deep-liquid asymptotes: rectangular √π = 1.77, upright cylinder √1.841 = 1.36 (×√2 on a diameter basis).

Horizontal-cylinder and elliptical curves use the equivalent-rectangle method (good to ~10–15% at mid fill; near-full and violent cases are where a VOF free-surface CFD run takes over).
