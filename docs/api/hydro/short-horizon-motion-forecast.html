Illustrative tracer   Synthetic JONSWAP sea · simplified analytic RAOs · deterministic phases. Numbers demonstrate the workflow, not a certified vessel model.

Honest horizon. The forecast is bounded by the predictable zone — a group-velocity wedge set by spectral bandwidth. Realistic short-crested seas give tens of seconds to ~1–2 min, not the 4–5 min sometimes marketed.

Two data modes. Measured — live vessel & structure motions from an onboard MMS / MRU (motion-reference unit) give the "now" ground truth. Predicted — a phase-resolved wave forecast transferred through the asset RAO extends it across the predictable zone. Both feed the same rolling go / no-go.

Where the value is. The wave-sensing and motion-sensing front-end (X-band radar / lidar / up-wave buoy → phase-resolved surface; MMS / MRU for measured motion) is partner/licensed. This layer is the asset-motion transfer, the operability decision, and the learning loop (measured MMS vs forecast) — on AQWA / OrcaFlex RAOs.
