## Convergence Logic

```python
# Read targets
targets = {
    'Line01': 80.0,  # kN
    'Line02': 80.0,
    # ... Line03-16
}

tolerances = {
    'Line01': 50,  # %
    'Line02': 50,
    # ... Line03-16
}

# Check convergence
for line in lines:
    diff_percent = abs(current[line] - targets[line]) / targets[line] * 100
    if diff_percent > tolerances[line]:
        return False  # Not converged
return True  # All lines converged
```
