## Usage

### For Single Load Case Analysis

Reference base files directly in main model:

```yaml
Environment:
  - includefile: base_files/_03a_sea_density.yml
  - includefile: base_files/_03b_seabed.yml
  - includefile: base_files/_03c_waves_base.yml
  - includefile: base_files/_03d_current_base.yml
  - includefile: base_files/_03e_wind_base.yml
```

Then manually edit the base environment files to set desired directions.

### For Operability Analysis (12 Load Cases)

1. **Base files** remain unchanged (single copy)
2. **Load case files** override only the direction parameters
3. **Batch script** programmatically applies load case overrides

See `../load_cases/` directory for load case-specific overrides.

## Maintenance

### Updating Wave Spectrum

To change the wave spectrum (e.g., switch to 10-year return period):

1. Edit **ONLY** `_03c_waves_base.yml`
2. Update wave parameters (Hs, Tp, components)
3. Change automatically applies to all 12 load cases

### Updating Current Profile

To modify current profile:

1. Edit **ONLY** `_03d_current_base.yml`
2. Update RefCurrentSpeed and depth profile
3. Change automatically applies to all 12 load cases

### Updating Wind Conditions

To change wind parameters:

1. Edit **ONLY** `_03e_wind_base.yml`
2. Update WindSpeed and spectrum settings
3. Change automatically applies to all 12 load cases
