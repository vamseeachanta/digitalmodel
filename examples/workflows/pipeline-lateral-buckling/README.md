# Pipeline Lateral Buckling Workflow

Screens HP/HT lateral buckling for a representative subsea pipeline using the existing pipeline buckling router.

Run:

```bash
uv run python -m digitalmodel examples/workflows/pipeline-lateral-buckling/input.yml
```

Expected outputs:

- `examples/workflows/pipeline-lateral-buckling/results/input.yml`
- `examples/workflows/pipeline-lateral-buckling/results/input_lateral_buckling.csv`
