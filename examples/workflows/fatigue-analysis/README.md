# Fatigue Analysis

Computes cumulative Miner fatigue damage from direct S-N cycle bins using a bundled DNV C2 seawater cathodic-protection curve row.

Run:

```bash
uv run python -m digitalmodel examples/workflows/fatigue-analysis/input.yml
```

Expected outputs are written to `examples/workflows/fatigue-analysis/results/`, including the resolved result configuration `input.yml`.
