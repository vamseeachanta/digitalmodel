# Pipeline Upheaval Buckling Workflow

Screens buried pipeline upheaval buckling using the existing pipeline buckling calculations and a bundled cover/download requirement.

Run:

```bash
uv run python -m digitalmodel examples/workflows/pipeline-upheaval-buckling/input.yml
```

Expected outputs:

- `examples/workflows/pipeline-upheaval-buckling/results/input.yml`
- `examples/workflows/pipeline-upheaval-buckling/results/input_upheaval_buckling.csv`
