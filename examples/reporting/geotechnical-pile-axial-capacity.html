Methodology

Axial pile capacity in clay using alpha method per API RP 2GEO

Standard: API RP 2GEO

Equation eq1: Alpha factor
$$\alpha = 0.5 \left(\frac{S_u}{\sigma'_v}\right)^{-0.5}$$
Adhesion factor for clay
(eq1)
Equation eq2: Unit skin friction
$$f = \alpha \cdot S_u$$
Unit skin friction capacity
(eq2)
Equation eq3: Total axial capacity
$$Q = \pi D \sum f_i L_i + N_c S_u A_p$$
Sum of skin friction + end bearing
(eq3)

Calculations

Step 1: Alpha factor calculation

Su/sigma_v' = 60/150 = 0.4 alpha = 0.5 * (0.4)^(-0.5) = 0.5 * 1.581 = 0.791
Ref: API RP 2GEO Sec 7.3

Step 2: Unit skin friction

f = alpha * Su = 0.791 * 60 = 47.43 kPa

Step 3: Total skin friction capacity

Qf = pi * D * f * L = pi * 1.0 * 47.43 * 30 = 4470.56 kN

Step 4: End bearing capacity

Nc = 9.0 (API RP 2GEO for deep piles) Ap = pi/4 * D^2 = 0.785 m^2 Qp = Nc * Su * Ap = 9.0 * 60 * 0.785 = 424.12 kN

Step 5: Total axial capacity

Q = Qf + Qp = 4470.56 + 424.12 = 4894.68 kN
Ref: API RP 2GEO Eq. 7.3-1
