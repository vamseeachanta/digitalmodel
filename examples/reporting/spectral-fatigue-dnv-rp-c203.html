Methodology

Spectral fatigue analysis using frequency-domain methods per DNV-RP-C203 Section 5. A stress response PSD is obtained from hydrodynamic transfer functions and a wave spectrum (e.g. JONSWAP). Fatigue damage is computed from spectral moments without generating time histories. Three methods are compared: 1. Narrow-band approximation (conservative upper bound) 2. Dirlik empirical method (accurate for broadband) 3. Tovo-Benasciutti weighted method For narrow-band processes the analytical damage rate is: D_dot = nu_0 * (2*sqrt(2)*sigma)^m * Gamma(1 + m/2) / A where nu_0 is the zero-crossing rate, sigma = sqrt(m0), and Gamma is the gamma function.

Calculations

Step 1: S-N curve verification

DNV-RP-C203 D-curve in air: A = 5.73e11 MPa^3 cycles, m = 3.0 At S = 100 MPa: N = 5.73e11 / 100^3 = 573,000 cycles At S = 200 MPa: N = 5.73e11 / 200^3 = 71,625 cycles CAFL = 52.63 MPa: below this, infinite life assumed.

Step 2: Miner's damage -- hand calculation example

Using a simple 3-block histogram (no CAFL, A = 1e12, m = 3.0): Block 1: S = 100 MPa, n = 1000 -> N = 1e6, D1 = 1.0e-3 Block 2: S = 200 MPa, n = 500 -> N = 1.25e5, D2 = 4.0e-3 Block 3: S = 50 MPa, n = 2000 -> N = 8e6, D3 = 2.5e-4 Total D = 5.25e-3 Safety factor = 1 / D = 190.5

Step 3: Narrow-band spectral fatigue

Given sigma = sqrt(m0) = 40 MPa (RMS stress), m = 3.0: Equivalent stress range = 2 * sigma = 80 MPa N_80 = 5.73e11 / 80^3 = 1,119,141 cycles With zero-crossing rate nu_0 = 0.1 Hz (10 s period): Damage per second = 0.1 / 1,119,141 = 8.94e-8 Damage per hour = 3.22e-4 Fatigue life = 1 / (8.94e-8) = 1.12e7 s = 129 days This is the conservative narrow-band estimate. Dirlik and Tovo-Benasciutti reduce damage for broadband processes by accounting for the bandwidth parameter alpha.

Step 4: Seawater environment adjustments

Seawater + cathodic protection (DNV-RP-C203 Table 2-2): A_sw_cp = A_air * 0.87 = 4.985e11 CAFL retained (CP suppresses corrosion below CAFL). Seawater free-corrosion (DNV-RP-C203 Cl. 2.4.3): A_sw_free = A_air * 0.72 = 4.126e11 CAFL removed (no fatigue limit in corrosive environment).

Step 5: Method comparison summary

For broadband stress PSD (offshore structural response): - Narrow-band: highest damage (conservative upper bound) - Dirlik: typically 40-70% of narrow-band for alpha < 0.7 - Tovo-Benasciutti: weighted blend, between NB and Dirlik Selection guidance per DNV-RP-C203: - Narrow-band: quick screening or narrow-banded response - Dirlik: general broadband spectral fatigue - Time-domain rainflow: definitive validation
