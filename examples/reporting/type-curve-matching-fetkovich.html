Methodology

Fetkovich type curve matching converts production rate-time data to dimensionless coordinates (qDd, tDd) for comparison against analytical decline stems. The dimensionless rate qDd = q/qi and dimensionless time tDd = Di*t allow production data to be overlaid on pre-computed type curves to identify the reservoir depletion character. Blasingame extends this by using material balance time tca = Np/q (cumulative divided by rate) which linearizes boundary-dominated flow and improves late-time matching for unconventional reservoirs. The b-factor is identified by nonlinear least-squares fitting of the Arps hyperbolic model q(t) = qi / (1 + b*Di*t)^(1/b) using scipy.optimize.curve_fit.

Assumptions
  1. Production data represents boundary-dominated flow
  2. Single-layer homogeneous reservoir (no commingled zones)
  3. Constant wellbore flowing pressure during decline period
  4. No operational shutdowns or rate changes during analysis period
  5. Arps hyperbolic model is appropriate (0 < b < 1)
