# GTM Demo Suite -- GIF Screencasts

Animated scroll-through previews of the five GTM demo reports plus a
cross-demo comparison matrix. Each GIF is 1024x640 at ~120 ms/frame.

## Deliverables

| File | Source | Description |
|------|--------|-------------|
| `demo_01_freespan.gif` | Demo 1 HTML report | DNV freespan / VIV screening (680 cases) |
| `demo_02_wall_thickness.gif` | Demo 2 HTML report | Multi-code wall thickness comparison (72 cases) |
| `demo_03_mudmat_installation.gif` | Demo 3 HTML report | Deepwater mudmat installation (180 cases) |
| `demo_04_shallow_pipelay.gif` | Demo 4 HTML report | Shallow water S-lay pipelay (60 cases) |
| `demo_05_jumper_installation.gif` | Demo 5 HTML report | Deepwater rigid jumper installation (300 cases) |
| `demo_comparison_matrix.gif` | Comparison JSON data | Vessel + structure capability matrix across all demos |

## How they were generated

- **Demo 1--5 GIFs:** Playwright headless browser captured scrolling
  screenshots of the validated HTML reports in `../output/`, then
  assembled into animated GIFs via Pillow.
- **Comparison matrix GIF:** Generated by `generate_comparison_matrix_gif.py`
  which renders `../results/vessel_comparison_matrix.json` and
  `../results/structure_comparison_matrix.json` into a styled HTML page
  and captures the same scrolling-screenshot workflow.

## Regenerating

```bash
cd digitalmodel
python3.12 examples/demos/gtm/media/generate_comparison_matrix_gif.py
```
