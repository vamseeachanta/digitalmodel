# GTM Demo Suite -- GIF Screencasts

Animated scroll-through previews of the five GTM demo reports plus a
cross-demo comparison matrix. Each GIF is 1024x640 at ~120 ms/frame.

## Deliverables

| File | Source | Description |
|------|--------|-------------|
| `demo_01_freespan.gif` | Demo 1 HTML report | DNV freespan / VIV screening (680 cases) |
| `demo_02_wall_thickness.gif` | Demo 2 HTML report | Multi-code wall thickness comparison (72 cases) |
| `demo_03_mudmat_installation.gif` | Demo 3 HTML report | Deepwater mudmat installation (180 cases) |
| `demo_04_shallow_pipelay.gif` | Demo 4 HTML report | Shallow water S-lay pipelay (60 cases) |
| `demo_05_jumper_installation.gif` | Demo 5 HTML report | Deepwater rigid jumper installation (300 cases) |
| `demo_comparison_matrix.gif` | Comparison JSON data | Vessel + structure capability matrix across all demos |
| `demo_01_freespan_workflow.gif` | Composed scenes + Demo 1 HTML report | End-to-end workflow: prompt → setup → execution → report → browser walkthrough |
| `demo_02_wall_thickness_workflow.gif` | Composed scenes + Demo 2 HTML report | End-to-end workflow: prompt → setup → execution → report → browser walkthrough |
| `demo_03_mudmat_installation_workflow.gif` | Composed scenes + Demo 3 HTML report | End-to-end workflow: prompt → setup → execution → report → browser walkthrough |
| `demo_04_shallow_pipelay_workflow.gif` | Composed scenes + Demo 4 HTML report | End-to-end workflow: prompt → setup → execution → report → browser walkthrough |

## How they were generated

- **Demo 1--5 GIFs:** Playwright headless browser captured scrolling
  screenshots of the validated HTML reports in `../output/`, then
  assembled into animated GIFs via Pillow.
- **Comparison matrix GIF:** Generated by `generate_comparison_matrix_gif.py`
  which renders `../results/vessel_comparison_matrix.json` and
  `../results/structure_comparison_matrix.json` into a styled HTML page
  and captures the same scrolling-screenshot workflow.
- **Demo 1 workflow GIF:** Generated by `generate_demo_01_workflow_gif.py`.
  Composes five staged scenes (prompt, setup, execution, report generation,
  browser walkthrough label) followed by a scroll-through of the validated
  Demo 1 HTML report. Part of the workflow-style upgrade tracked in issue #2288.
- **Demo 2 workflow GIF:** Generated by `generate_demo_02_workflow_gif.py`.
  Composes five staged scenes (prompt, setup, execution, report generation,
  browser walkthrough label) followed by a scroll-through of the validated
  Demo 2 HTML report. Shows the full end-to-end story arc for issue #1809.
- **Demo 3 workflow GIF:** Generated by `generate_demo_03_workflow_gif.py`.
  Composes five staged scenes (prompt, setup, execution, report generation,
  browser walkthrough label) followed by a scroll-through of the validated
  Demo 3 HTML report. Part of the workflow-style upgrade tracked in issue #2288.
- **Demo 4 workflow GIF:** Generated by `generate_demo_04_workflow_gif.py`.
  Composes five staged scenes (prompt, setup, execution, report generation,
  browser walkthrough label) followed by a scroll-through of the validated
  Demo 4 HTML report. Part of the workflow-style upgrade tracked in issue #2288.
